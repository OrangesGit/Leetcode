# 406. Queue Reconstruction by Height

## Greedy

```java
class Solution {
    public int[][] reconstructQueue(int[][] people) {
        Arrays.sort(people, (a,b)->{
            if(a[0] == b[0]) return a[1] - b[1];
            return b[0] - a[0];
        });
        List<int[]> que = new ArrayList<>();
        for(int[] p : people)
            que.add(p[1],p);
        return que.toArray(new int[que.size()][]);
    }
}
```

$$
O(N)+(N)
$$
