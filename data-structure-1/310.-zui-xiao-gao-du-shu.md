# 310. 最小高度树

## 对每个节点BFS（超时）

```java
class Solution {
    public List<Integer> findMinHeightTrees(int n, int[][] edges) {
        List<List<Integer>> graph = new ArrayList();
        for(int i=0; i<n; i++) graph.add(i, new ArrayList());
        for(int[] edg:edges){
            int node1 = edg[0], node2 = edg[1];
            graph.get(node1).add(node2);
            graph.get(node2).add(node1);
        }
        List<Integer> res = new ArrayList();
        int[] heights = new int[n];
        for(int i=0; i<n; i++){
            boolean[] visited = new boolean[n];
            Queue<Integer> queue = new LinkedList<>();
            queue.offer(i);
            int height = 0;
            while(!queue.isEmpty()){
                
                height += 1;
                int size = queue.size();
                for(int j=0; j<size; j++){
                    int currNode = queue.poll();
                    visited[currNode] = true;
                    for(int neighbor:graph.get(currNode)){
                    if(!visited[neighbor]){
                        queue.offer(neighbor);
                    }
                }
                }

            }
            heights[i] = height;
        }
        int min = Integer.MAX_VALUE;
        for(int i:heights) min = Math.min(min, i);
        for(int i=0; i<n; i++){
            if(heights[i] == min) res.add(i);
        }
        return res;
    }
}
```

## 从leaf节点出发剪枝

```java
class Solution {
    public List<Integer> findMinHeightTrees(int n, int[][] edges) {
        if (n < 2) {
            ArrayList<Integer> centroids = new ArrayList<>();
            for (int i = 0; i < n; i++)
                centroids.add(i);
            return centroids;
        }
        List<List<Integer>> graph = new ArrayList();
        for(int i=0; i<n; i++) graph.add(new ArrayList());
        for(int[] edge:edges){
            graph.get(edge[0]).add(edge[1]);
            graph.get(edge[1]).add(edge[0]);
        }
        List<Integer> leaves = new ArrayList();
        for(int i=0; i<n; i++){
            if(graph.get(i).size() == 1)
                leaves.add(i);
        }
        
        int remainNodes = n;
        while(remainNodes>2){
            remainNodes -= leaves.size();
            List<Integer> newLeaves = new ArrayList();
            for(int leaf:leaves){
                Integer neighbor = graph.get(leaf).get(0);
                graph.get(neighbor).remove((Integer)leaf);
                if(graph.get(neighbor).size() == 1)
                    newLeaves.add(neighbor);
            }
            leaves = newLeaves;
        }
        return leaves;
    }
}
```

O(V)+O(V)
