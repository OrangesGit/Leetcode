# 1. Find the Length of a Linked List

```java
    //This is a member function of a SinglyLinkedList class
    //Class does not have a data-member size
    public int length() {
      int count = 0;
      // Write -- Your -- Code
      Node curr = headNode;
      while(curr != null){
        curr = curr.nextNode;
        count++;
      }
      return count;
    }
}
```
